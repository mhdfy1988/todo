# 代办：三种视觉风格对比 v0.2

> 状态：视觉方向探索稿  
> 目标：在不改变产品功能的前提下，比较三种适合长期桌面使用的外观方向  
> 交互入口：[`prototype/themes/index.html`](../prototype/themes/index.html)

> 2026-07-13 决策：三套设计全部保留，极简原生风作为当前主方案优先完成；最终口径见[《视觉方向决策 v0.3》](./视觉方向决策-v0.3.md)。本文件继续保留当时的中立比较记录。

## 1. 对比方法

三套设计使用完全相同的内容和状态：

- 当前任务：整理桌面待办设计方案。
- 今日进度：4 / 7。
- 金币余额：12。
- 三项后续任务。
- 完成、暂时做不了、快速新增。
- 右下角收起态和完整展开态。

因此需要比较的是长期观感、桌面适配、宠物表现和信息可读性，不是功能差异。

## 2. 方向一：极简原生风

### 定位

像系统原生桌面组件一样安静可靠。任务面板尽量退后，宠物是唯一明显的活泼元素。

### 视觉特征

- 浅色中性灰与细腻层次。
- 接近 Windows 桌面组件的清晰结构。
- 较小圆角、轻阴影和低饱和强调色。
- 高信息密度但不过度拥挤。
- 动效短暂，适合长期常驻。

### 优点

- 容易融入多数工作桌面。
- 任务文字最清晰，长时间使用压力较小。
- 后续扩展历史、设置和备份页面比较自然。

### 风险

- 如果处理得太保守，可能缺少产品记忆点。
- 宠物造型和完成反馈必须承担主要个性表达。

## 3. 方向二：深色透明风

### 定位

适合深色桌面和夜间环境，强调悬浮、轻盈和克制的科技感。

### 视觉特征

- 黑灰半透明玻璃面板。
- 冷色微光与局部高亮，不使用俗套紫色渐变。
- 背景模糊、细边框和低强度光晕。
- 状态、数字和金币更像桌面状态组件。
- 宠物拥有微光轮廓，但仍保持亲切。

### 优点

- 悬浮感强，和深色壁纸搭配较好。
- 收起态体积小，视觉存在感可控。
- 宠物动画和金币反馈更容易产生精致效果。

### 风险

- 浅色壁纸下可能需要额外遮罩保证可读性。
- 透明、模糊和阴影效果需要关注桌面性能。
- 长文本页面不能全部采用玻璃效果。

## 4. 方向三：像素桌宠风

### 定位

把宠物养成的趣味明确表达出来，但任务面板仍保持克制，避免变成复杂游戏界面。

### 视觉特征

- 像素宠物、像素金币和清晰的网格边缘。
- 紧凑任务卡与轻量游戏提示。
- 少量高对比强调色。
- 完成任务时使用一帧明确的像素反馈。
- 展开页仍以任务阅读为主，不堆叠游戏数值。

### 优点

- 产品记忆点最强。
- 宠物成长和外观变化更容易持续扩展。
- 小尺寸收起态依然容易识别。

### 风险

- 用户对像素审美的接受度差异较大。
- 游戏感过重会削弱工作工具的可靠感。
- 中文字号过小时容易和像素元素互相干扰。

## 5. 建议评审维度

查看三套原型时，优先判断：

1. 哪一套愿意每天挂在桌面上八小时？
2. 哪一套在右下角收起后最不打扰？
3. 哪一套展开后最容易立即看清当前任务？
4. 哪一套宠物让人愿意回来喂食和观察成长？
5. 哪一套放到工作环境中不会显得过于幼稚？
6. 是否希望主体风格和宠物风格混搭？

不要求三选一。最终方案可以采用“极简原生任务面板 + 像素宠物”或“深色透明收起态 + 原生风完整窗口”等组合，但同一个状态内需要保持一致，避免视觉语言混乱。
