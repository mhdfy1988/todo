# Changelog

本文件记录“待办”面向使用者的重要变更。

格式遵循 [Keep a Changelog 2.0.0](https://keepachangelog.com/en/2.0.0/)，版本号遵循[语义化版本](https://semver.org/lang/zh-CN/)。

## [Unreleased]

## [0.1.4] - 2026-07-24

### Changed

- 将用户可见的软件名称由“代办”更正为“待办”，同时保留既有 `zuoban` 数据目录、包名和应用标识。
- 将“完成记录”页面简化为“已完成”：返回入口和页名合并进顶部窗口标题栏，“复制本周完成”移入该页右上角更多菜单。
- 将待办页底部的“已完成”、数量和箭头收成右侧紧凑按钮，三者都可点击，同时保留整行其余区域不可点击以避免误触。
- 将软件图标由宠物形象改为深色底座与珊瑚橙任务牌组成的“悬浮任务牌”，并统一更新桌面、托盘与安装器图标资源。

### Fixed

- 修复子代办删除入口隐藏后，剩余排序柄横向移位的问题；可选操作统一收进靠右自适应操作区。
- 修复展开子代办时父行首个网格行缩到控件固有高度，导致父待办整行向上跳动的问题；当前待办左侧橙色标记也会随普通态和编辑态首行保持居中。

## [0.1.3] - 2026-07-24

### Changed

- 将子代办新增入口固定在父待办行右侧，进度数字只负责展开或收起，并移除子列表底部重复的“添加子代办”入口。
- 父待办进入标题或期限修改态、子代办进入新增态时，继续保留右侧操作列位置，避免删除与排序按钮横向跳动。

### Fixed

- 修复没有子代办的父项取消空白新增后仍残留展开空区的问题；空白失焦或按 `Esc` 会恢复收起，已有子项的父项取消新增时仍保持展开。
- 修复两个连接同时首次打开同一账本时，SQLite 切换 WAL 日志模式可能因瞬时锁竞争而启动失败的问题；只对明确的 busy/locked 做有界重试，并校验实际日志模式。

## [0.1.2] - 2026-07-24

### Added

- 新增一级子代办：支持添加、改名、完成、撤销、软删除和同一父项内重排。
- 搜索结果保留父项上下文；完成记录按父项分组折叠；桌面胶囊显示当前子动作和父项进度。
- 本地账本升级到 schema v5，并在迁移正式数据前生成和验证一致性备份。

### Changed

- 子代办新增与标题修改支持失焦保存；新增成功后恢复底部“添加子代办”，空白失焦直接取消。
- 勾选父待办时，会在同一事务内完成全部未完成子项和父项；子项不发金币，父项仍只奖励 1 枚金币。
- 周回顾在父项于本周完成时，可以带入该父项下更早完成且仍然有效的子项。

### Fixed

- 修复上次在展开面板退出后，冷启动胶囊直接复用展开窗口左上角坐标而出现在错误位置的问题；启动时会按胶囊尺寸归位到原显示器工作区右下角。

## [0.1.1] - 2026-07-22

### Added

- 新增“复制本周完成”，把当前自然周内仍然有效的完成项复制为 Markdown 编号列表。

### Fixed

- 发布流程拒绝覆盖已经公开的同版本安装器、更新签名和 `latest.json`。

## [0.1.0] - 2026-07-22

### Added

- 首次发布 Windows x64 桌面悬浮待办。
- 支持快速记录、任意待办勾选完成、拖动或键盘重排、标题修改、软删除和可选期限。
- 支持完成记录与撤销、当前页面搜索、桌面胶囊、系统托盘和应用内更新。
- 使用本地 SQLite 事件账本保存任务、完成历史、奖励和幂等命令回执。
